# Analyse Statistique — Comportement des utilisateurs Instagram

## 📌 Objectif
Analyser le comportement des utilisateurs sur Instagram à travers 
des tests statistiques pour identifier des patterns significatifs.

## 🔍 Analyses réalisées
- Calcul et interprétation de la p-value
- Tests d'hypothèses (H0 / H1)
- Analyse des distributions
- Visualisation des résultats

## 🛠️ Technologies utilisées
- Python 3
- Jupyter Notebook
- Pandas
- Scipy / Statsmodels
- Matplotlib / Seaborn
